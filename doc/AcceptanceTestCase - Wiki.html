[edit] ATC

[edit] 001 First signup and signup successfully

  • Precondition
    • Database is created well and the communication function between server and client is also well
  • Expect Result
    • After you creating a personal account, you can check the information in database.
  • Steps
  1. AS shown in when you run this program, by typing Signup button on the above, enter the Signup screen
  2. After you enter the Signup screen, you have to fill in the blanks to signup
  3. When you finish the above two steps, click the Signup button, if you were the first time to signup and every blanks were filled in right way, then you will can see the information to tell you that you are succeed to register

[edit] 002 Signup using existed username

  • Precondition
    • You already have a registered username and the other three blanks you typed are well
  • Expect Result
    • To remind to you that the username already existed
  • Steps
  1. The steps are the same as 001, but the username is used to signup before

[edit] 003 Signup using wrong type of email address

  • Precondition
    • The other three blanks that you typed are well
  • Expect Result
    • To remind to you that your type of email address is wrong
  • Steps
  1. The steps are the same as 002, but because of your mistake, when you typed the email address, it is not in a correct type, for example, type a phone number instead of an email address


[edit] 004 Password is different from the verified password

  • Precondition
    • The blanks of username and email address are filled in well
  • Expect Result
    • To remind to you that the password you type two times are different
  • Steps
  1. The steps are the same as 003, but mostly, the verified password you type is different from the password that you typed.


[edit] 005 Missing blanks to fill in

  • Precondition
    • Forget to fill in one or more filed in the blanket
  • Expect Result
    • To tell you that which field you forget to fill in
  • Steps
  1. During the process of filling in the blanks, if you did not finish and tap the Sign up button, then it will remind to you which field you are missing


[edit] 006 Sign in successful

  • Precondition
    • You type the username and the password correctly
  • Expect Result
    • Jump to join-Krowdit screen
  • Steps
  1. type your username and password, then type sign in button

[edit] 007 Missing fields to sign in

  • Precondition
    • Missing either username or password to sign in
  • Expect Result
    • To tell you which field you are missing
  • Steps
  1. Switch to Sign in Screen, if you forget either username or password and try to sign in


[edit] 008 Sign in error

  • Precondition
    • Either username or password not found
  • Expect Result
    • To tell the user username or password error
  • Steps
  1. When you type the username or password, because of careless, some information is wrong, then you type the button Sign in

[edit] 009 very first time Sign in with FB

  • Precondition
    • net with VPN is OK
    • right FB UserName and Pwd
  • Expect Result
    • switch to main tab
  • Steps
  1. press Sign in with FB button
  2. input the UserName or Pwd
  3. press Okay button

[edit] 010 Sign in with FB TODO

  • Precondition
    • net with VPN is bad
    • right FB UserName or Pwd
  • Expect Result
    • pop up a window to tell user net is bad
  • Steps
  1. press Sign in with FB button


[edit] 011 after successful Sign in with FB

  • Precondition
    • net with VPN is good
    • had successful Sign in with FB previously
  • Expect Result
    • switch to main tab
  • Steps
  1. press Sign in with FB button
  2. press Okay button


[edit] Single Cell Expansion

[edit] 012 First Tap the Cell

  • Precondition
    • All Cells are narrow,the images inside it are concealed
    • The default color of the cells are red
  • Expect Result
    • The images are shown
    • The taped cell become wider and its color become green
  • Steps
  1. Tap one of the cells

[edit] 013 Become Back

  • Precondition
    • The cell become wider,the images inside it are shown
    • The color of the cell is green
  • Expect Result
    • The tapped cell becomes narrow,the images inside it are concealed
    • The color of the tapped cell become red
  • Steps
  1. Tap the wider cell

[edit] 014 Tap Another Cell

  • Precondition
    • One cell is tapped,it is wider,the images inside it are shown
    • The color of the cell is green
    • Other cells are narrow,their images are concealed,their color are red
  • Expect Result
    • The new tapped cell becomes wider,the images inside it are shown
    • The color of the new tapped cell become green
    • The previous one become narrow,red and images inside it are concealed
  • Steps
  1. Tap a new cell

[edit] 015 Scroll

  • Precondition
    • All Cells are narrow,the images inside it are concealed
    • The default color of the cells are red
  • Expect Result
    • The same as Precondition
  • Steps
  1. Tap a cell,scroll down,then leave the cell

[edit] 016 Enter Image

  • Precondition
    • If one of the cells is taped,the images are shown
    • The taped cell become wider and its color become green
  • Expect Result
    • We can see the information of the image
  • Steps
  1. Tap the logo of the image

[edit] 017 No Internet

  • Precondition
    • All Cells are narrow,the images inside it are concealed
    • The default color of the cells are red
    • The local Internet are not available
  • Expect Result
    • There are no images,only some marks,if you tap the mark you can not join the krowd
    • The taped cell become wider and its color become green
  • Steps
  1. Tap one of the cells

[edit] 018 The Internet are Slow

  • Precondition
    • All Cells are narrow,the images inside it are concealed
    • The default color of the cells are red
    • The speed of local Internet is slow
  • Expect Result
    • The images are shown,you can join the krowd
    • The taped cell become wider and its color become green
  • Steps
  1. Tap one of the cells

[edit] 019 The Internet is OK

  • Precondition
    • All Cells are narrow,the images inside it are concealed
    • The default color of the cells are red
    • The speed of local Internet is OK
  • Expect Result
    • The images are shown,you can join the krowd
    • The taped cell become wider and its color become green
  • Steps
  1. Tap one of the cells

[edit] KrowdList

[edit] 020 Lists

  • Precondition
    • Input a given coordinates
  • Expect Result
    • List the first 25 krowd lists
  • Steps
  1. Tap "Login" Button

[edit] 021 Search

  • Precondition
    • The radius is less than a given value
  • Expect Result
    • Enlarge the radius to search until find a krowd
  • Steps
  1. Tap "Login" Button

[edit] 022 Another List

  • Precondition
    • The first 25 krowds are already listed
  • Expect Result
    • Enlarge the radius to search until find a krowd
    • Another 25 krowd lists are shown
  • Steps
  1. Scroll down the screen

[edit] 023 Keep List

  • Precondition
    • Input a given coordinates
  • Expect Result
    • The list from 0 to 24 are keep
  • Steps
  1. Join a krowd is less than 30 seconds and leave

[edit] 024 Refresh

  • Precondition
    • Input a given coordinates
  • Expect Result
    • Refresh the sorting list
  • Steps
  1. Join a krowd is greater than 30 seconds and leave

[edit] 025 Release

  • Precondition
    • There are 200 sorting results
  • Expect Result
    • The lists from 25 to 49 are list
  • Steps
  1. Scroll the screen

[edit] 026 Change The Coordinates

  • Precondition
    • Input another coordinates
  • Expect Result
    • List the first 25 krowd lists
  • Steps
  1. Tap "Login" Button

  • Expect Result

  • Steps

[edit] Password Recovery

[edit] 037 Has An Account

  • Precondition
    • The user has an account
  • Expect Result
    • Email the user with registered password
  • Steps
  1. Tap "Forget Password" button,it will show an alert-view
  2. Input the registered email and "OK" button on the alert-view

[edit] 038 Internet Problem

  • Precondition
    • The user has an account
  • Expect Result
    • There is an alert-view to show that the Internet has some problem
  • Steps
  1. Tap "Forget Password" button,it will show an alert-view
  2. Input the registered email and "OK" button on the alert-view
  3. The Internet is not available

[edit] 039 No Account

  • Precondition
    • The user does not have an account
  • Expect Result
    • There is an alert-view to show that the use does not have an account
    • Ask the user whether use this email to register.If yes,register,other wise quit the alert-view
  • Steps
  1. Tap "Forget Password" button,it will show an alert-view
  2. Input the registered email and "OK" button on the alert-view

[edit] Jak Search

[edit] 040 Search 1

  • Precondition

  • Expect Result
    • Show the sorting result,server correspond to location,client correspond to time
    • It includes Administrator Logo
    • The user can tap the "Krowd" button to join the krowd
  • Steps
  1. Tap "searchBar" button,show search field
  2. Input required information

[edit] 040 Search 2

  • Precondition
    • Server does not have searched information
  • Expect Result
    • The screen show there is no such information
  • Steps
  1. Tap "searchBar" button,show search field
  2. Input required information
