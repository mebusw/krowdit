  • How to setup shortcuts in terminal
    • Save hfcca.py to /Application, then type below 
$ ln -s  /Applications/hfcca.py  /usr/bin/hfcca
    • copy doxygen to /Application, then 
$ ln -s  /Applications/Doxygen.app/Contents/Resources/doxygen
  • How to turn on color mode in terminal
    • Edit your ~/.bash_profile, add below lines 
export CLICOLOR=1
export LSCOLORS=ExFxCxDxBxegedabagacad
    • and then type below in terminal 
$ source ~/.bash_profile
