[edit] compile iphone project in terminal

  • help commands 
xcodebuild -h
xcodebuild -version
xcodebuild -list
  • compiling commands (under the project folder) 
xcodebuild

or with arguments 

xcodebuild -sdk iphonesimulator4.3
xcodebuild clean -sdk iphonesimulator4.3 -configuration=Debug
  • compiling remotely ( without unlocking keychain, it will be failed with reason "User interaction is not allowed.") 
ssh ssh jacky@192.168.1.105
security unlock-keychain
xcodebuild
  • No password input in scipts

generate keys in local machine 

ssh-keygen -t rsa

copy ~/.ssh/id_rsa.pub to remote machine that you want to connect on the remote machine 

mv id_rsa.pub ~/.ssh/authorized_keys
  • Remember to modify the /etc/sshd_config or /etc/ssh/sshd_config of the remote machine 
PasswordAuthentication yes
